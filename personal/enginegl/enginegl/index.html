EngineGL

C++ - OpenGL - Shadowmap - Envmapping


This is a work in progress toy renderer written in C++ and OpenGL to learn new graphics techniques and C++. The renderer features at the moment:

  • Forward rendering
  • Spot and directional lights support
  • Shadowmapping
    • Soft edge shadows
    • Shadow support for multiple lights
  • Environment mapping + simple environment reflections.

Soft shadows

To add soft shadows I implemented the technique described in GPU Gems 2 ch. 17.
Basically it adds some noise to offset the sample points for the PCF technique. This removes the patterns produced by the original PCF in exchange for noise, which is much more pleasing to the eye:
